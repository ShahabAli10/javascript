Rules for naming JS variable

  • Variable names can only contain letters, digits, underscores, and dollar signs. For example: $my_1stVariable
  • Variables must begin with a letter, underscore, or dollar sign. For example: $name, _name, or name
  • Variable names are case sensitive.
  • Variable names should not be JS reserved keywords.

Example of declaring 3 variables in one statement

Here is how to declare three variables in one statement:

let var1 = 1, var2 = 2, var3 = 3;

Legal and Illegal Variable Names

5 Legal Variable Names:

  • myVariable
  • _myVariable
  • $myVariable
  • var123
  • first_Name

5 Illegal Variable Names:

  • 123var (Cannot start with a number)
  • my-variable (Hyphens are not allowed)
  • var!name (Special characters like "!" are not allowed)
  • if (Cannot use JS reserved keywords)
  • my variable (Spaces are not allowed)
